Data Ajax

A basic AjaxLoader wrapper that inserts AJAXed in content.

Insert content after an element (data-ajax-after)

Example

Hello World

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Cupiditate, dolorem explicabo perferendis vitae magnam.

Hello World - Part 2

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Cupiditate, dolorem explicabo perferendis vitae magnam.

View code 
<section data-ajax-after="ajax/data-ajax-extra-en.html" class="original">
	<h4 id="anchor1">Hello World</h4>
	<p>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Cupiditate, dolorem explicabo perferendis vitae magnam.</p>
</section>
<section data-ajax-after="ajax/data-ajax-extra-2-en.html" class="original">
	<h4 id="anchor1">Hello World - Part 2</h4>
	<p>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Cupiditate, dolorem explicabo perferendis vitae magnam.</p>
</section>
